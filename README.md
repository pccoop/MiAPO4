# MiAPO4
Репозиторий для коллективной работы для лабораторной работы 4 по МиАПО 
Итак, ход дальнейшей работы: 
	1. Необходимо склонировать репозиторий командой clone на свое устройство.
	2. Перейдите в репозиторий командой cd
	3. Необходимо создать новую ветку по СВОЕЙ ФАМИЛИИ (по списку потом проверим ;D ) командой branch и перейти на нее командой checkout.
	4. В своей ветке вы создаете текстовый файл (в названии фамилия и расширение либо '.md', либо '.txt'), в котором будет содержаться переделанный код от того самого всеми любимого калькулятора.
	5. После сохранения текстового файла, запушивайте изменения в репозиторий (git push -u origin 'название вашей ветки по ФАМИЛИИ') и после рассмотрения внесенных на вашей ветке изменений, соединим вашу ветку с основной, в которой в будущем и отобразится ваш файл.
	6. Приятной работы, господа ИС-235 <3
